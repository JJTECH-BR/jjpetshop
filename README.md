# 🐾 JJ Pet Shop - Landing Page

Uma landing page moderna e responsiva desenvolvida para apresentar serviços de banho, tosa e estética animal. O projeto foca em uma interface limpa e amigável para atrair tutores de pets, exibindo detalhes dos serviços, pacotes de preços, galeria de fotos e um formulário para agendamento de visitas.

## 🚀 Funcionalidades

* **Apresentação de Serviços:** Cards visuais e detalhados com os cuidados oferecidos (banho, tosa, corte de unhas, limpeza de ouvidos, etc.).
* **Seção de Pacotes e Preços:** Tabela de preços estruturada (Essencial, Completo, Premium) com destaque visual para o plano mais procurado.
* **Formulário de Agendamento:** Interface de contato para que os clientes solicitem serviços informando dados básicos e o serviço desejado.
* **Design Responsivo:** Estrutura fluida que se adapta perfeitamente a dispositivos móveis, tablets e desktops, incluindo um menu *toggle* (hambúrguer) para telas menores.
* **Navegação Âncora (One-Page):** Links no cabeçalho que direcionam o usuário de forma suave para as respectivas seções da página.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica e acessível do conteúdo.
* **CSS3:** Estilização de componentes, layouts e uso de variáveis de cor (ex: `--card-bg`).
* **JavaScript:** Scripts (`script.js`) voltados para a interatividade da página, como a abertura do menu mobile e a manipulação do envio do formulário de contato.
* **Google Fonts:** Tipografia moderna utilizando as famílias *Inter* para textos corridos e *Plus Jakarta Sans* para títulos.
